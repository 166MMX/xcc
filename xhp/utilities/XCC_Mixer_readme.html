XCC Mixer

This is the MIX manager for C&C: Tiberian Sun and C&C: Red Alert 2. It also works for C&C: Tiberium Dawn and C&C: Red Alert and all expansion packs. It let's you view the contents of all MIX files and let's you extract any file you want.

An installation is not required. However, this program places some keys in the registry.

If you get an error about a missing DLL, download the XCC DLL Pack. Replace the missing DLL in your windows system directory if necessary.

If you've got a question about this program, please visit the XCC Home Page and post a message in the forum.
