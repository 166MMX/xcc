Guide: Using XCC Map Encoder/Decoder

Requirements

You will need XCC Map Encoder/Decoderr and Command & Conquer: Tiberian Sun or Red Alert 2.
You can download the XCC utilities from http://xhp.xwis.net/.

Description

This utility encodes TS/RA2 maps to a format that's 3x smaller then ZIP.

Create MMX

Creates an MMX file instead of a MPR file. This is only possible for RA2. MMX files don't cause a delay when RA2 is loaded, but auto-transfer doesn't work.

Create XMMF

Creates an XMMF file, which contains the MMX or MPR file. This provides far better compression and should be used when distributing your map over the internet. Include the decoder in the ZIP or upload it to the same place as your map, as future decoders might not be compatible.

Encode

Allows you to select the input files. Drag and drop can be used too.

Decode

Allows you to select the input files. Drag and drop can be used too.
