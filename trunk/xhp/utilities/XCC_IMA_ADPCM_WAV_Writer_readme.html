XCC IMA ADPCM WAV Writer Plug-In

This is a plug-in for WinAmp that allows you to write IMA ADPCM WAV files, which can be used for C&C: Red Alert 2.

You install this plug-in by starting XCC IMA ADPCM WAV Writer.exe.
You enable this plug-in by pressing Ctrl-P in WinAmp, selecting Plug-ins->Output->XCC IMA ADPCM WAV Writer Plug-In

You disable this plug-in by pressing Ctrl-P in WinAmp, selecting Plug-ins->Output->Nullsoft waveOut plug-in
