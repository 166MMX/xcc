XCC AUD Writer Plug-In

This is a plug-in for WinAmp that allows you to write AUD files, which can be used for C&C: Tiberium Dawn, C&C: Red Alert and C&C: Tiberian Sun and other games.

You install this plug-in by starting XCC AUD Writer.exe.
You enable this plug-in by pressing Ctrl-P in WinAmp, selecting Plug-ins->Output->XCC AUD Writer Plug-In

You disable this plug-in by pressing Ctrl-P in WinAmp, selecting Plug-ins->Output->Nullsoft waveOut plug-in

If you've got a question about this program, please visit the XCC Home Page and post a message in the forum.
