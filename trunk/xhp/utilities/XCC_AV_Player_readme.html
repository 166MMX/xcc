XCC AV Player

This is the Audio Video player for C&C: Tiberium Dawn, C&C: Red Alert, C&C: Tiberian Sun and C&C: Red Alert 2 and all expansion packs. It let's you play the AUD/WAV (audio) and VQA (video) files and convert the AUD files to WAV.

An installation is not required. However, this program places some keys in the registry.

If you get an error about a missing DLL, download the XCC DLL Pack. Replace the missing DLL in your windows system directory if necessary.
